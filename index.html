Request Header Parser Microservice

API Usage

This service parses your GET request header and returns your IP address, language, operating system and browser specs.

Example Usage:

https://microservice-header-parser-mseong.netlify.com/api/whoami

Example Output:

{"ipaddress":"113.211.209.107",
"language":"en-US,en;q=0.5",
"software":"Mozilla/5.0 (X11; Ubuntu; Linux x86_64; rv:50.0) Gecko/20100101 Firefox/50.0"}
